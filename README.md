# ☕ Programação Orientada a Objetos - Java

Repositório destinado ao armazenamento de projetos, exercícios e estudos práticos de **POO** desenvolvidos durante o curso de **Sistemas de Informação**.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Java (JDK 17+)
* **IDE:** Eclipse
* **Sistema Operacional:** Linux (Lubuntu)

## 📖 Conteúdo do Repositório
Atualmente focado em conceitos fundamentais de POO:
- [x] **Encapsulamento:** Uso de modificadores de acesso (`private`, `public`) e métodos Getters/Setters.
- [x] **Construtores:** Sobrecarga de construtores para criação opcional de objetos.
- [x] **Membros Estáticos:** Uso de `static` para identificadores únicos de instâncias.
- [ ] **Herança e Polimorfismo** (Próximos passos)

## 🚀 Como executar os projetos
1. Clone este repositório.
2. Importe o projeto no Eclipse via `File > Import > Existing Projects into Workspace`.
3. Localize a classe com o método `main` e execute.
